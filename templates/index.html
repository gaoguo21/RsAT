Free Online RNA-seq Analysis Tools

This free online RNA-seq analysis platform allows you to perform differential gene expression, pathway enrichment, ssGSEA and gene ID conversion directly in your browser. It also provides executable files in the Sources tab for converting FASTQ files to gene counts. They are designed and optimized to run locally on a personal computer.

DEG - Differential Gene Expression Analysis

Identify genes that are significantly differentially expressed between experimental conditions. This tool compares expression levels, calculates fold changes, p-values and adjusted pValues.

Learn how to use

Pathway Analysis

Analyze biological pathways enriched in your gene list to understand functional meaning. This tool links differential expression results to pathways, processes, and molecular functions for interpretation.

Learn how to use

ID2Symbol - Mapping gene ID to symbol

A tool that maps Ensembl Gene IDs (ENSG) and Entrez Gene IDs to official gene symbols. It enables standardized gene annotation and conversion between identifier systems.

Learn how to use

ssGSEA-single sample Gene Set Enrichment Analysis

A tool that calculate an enrichment score for a predefined gene set. It ranks all genes in a sample by expression and derives a score reflecting the relative enrichment of the predefined gene set.
